# PWM Generator (OrCAD X)

This project implements a Pulse Width Modulation (PWM) generator designed and simulated using OrCAD X. The circuit generates a PWM signal with adjustable duty cycle, which can be used for applications such as motor speed control or LED dimming

The design includes the main components required to generate a periodic signal and control the duty cycle. Simulations were performed in OrCAD X to verify the waveform, frequency, and duty cycle behavior of the PWM output.

The project focuses on analog circuit design and simulation, demonstrating how PWM signals can be generated and analyzed using electronic design automation tools.
